# eLearning Camp Frontend

This package will contain the React frontend for the eLearning Camp platform.

## Coming Soon

The frontend implementation will be a modern React application built with:
- Vite or Next.js
- TypeScript
- Modern React patterns and hooks
- Tailwind CSS
- API integration with the backend

## Development

Once implemented, you'll be able to start the frontend with:

```bash
pnpm dev
```
